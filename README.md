SubwayViewTest
==============

实现城市地铁站点查看的一个demo，支持多指缩放、站点选择监听，站点数据采用json格式

最近帮深圳气象局的朋友做的“我的都市天气”的“地铁天气”的模块，有类似功能需求的朋友可以参考下。

主要是要细心，没太大难度，缩放或滑动时以中心点为基准点计算每个站点的坐标。
